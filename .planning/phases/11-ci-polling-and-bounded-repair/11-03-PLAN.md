Plan 11-02 must provide:
```python
@dataclass(frozen=True)
class RepairBudget:
    max_attempts: int = 0

def decide_repair(trigger: RepairTrigger, budget: RepairBudget) -> RepairDecision: ...
def build_repair_task(orig_task: Task, trigger: RepairTrigger, decision: RepairDecision) -> Task: ...
def make_repair_attempt_requested_event(trigger: RepairTrigger, decision: RepairDecision) -> PipelineTaskEvent: ...
def make_repair_escalated_event(trigger: RepairTrigger, decision: RepairDecision) -> PipelineTaskEvent: ...
```
</interfaces>

Locked decisions:
- No continuous hidden polling, no daemon, no auto-merge, and no production recovery claims.
- CI polling must be explicit configured verification or configured sink metadata.
- Repair defaults to disabled or zero attempts unless configured.
- Repair must not call `release_task()` or remote `release()` to retry the same failed task.
- Repair task IDs must be deterministic and shaped as `<orig-task-id>-repair-<N>`.
- Repair tasks must not depend on the failed original task.
- Exhaustion emits `repair.escalated`.
</context>
