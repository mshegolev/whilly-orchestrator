# Whilly Capability Taxonomy

32 capabilities across 6 clusters. Each capability maps to exactly one spec file at
`openspec/specs/<slug>/spec.md`. This index is the authoritative reference for BASE-01.

## Naming Convention

Slugs MUST use kebab-case (e.g., `task-model-fsm`, `cli-surface`).
The slug is the directory name under `openspec/specs/` AND the spec ID returned by
`openspec spec list`. Each slug maps to exactly one future `spec.md`.
See `openspec/AUTHORING.md` for the full spec format and validation rules.
